# Design summary

- Design concept: Dark developer console
- Layout personality: compact asymmetric console
- Typography: JetBrains Mono headings with Inter body text
- Color palette: primary #020617, secondary #22c55e, accent #38bdf8, background #0b1120, text #e5e7eb, muted #94a3b8
- Shape language: Sharp corners
- Visual motif: terminal frames and code-like dividers
- Unique layout choices: first sections use Sprint console, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
